# 法會錄 ── Post-mortem Report

複雜科儀結束後，自動生成法會錄。記錄成敗、提煉經驗、為下次科儀提供改善依據。

---

## 何時撰寫法會錄

- 科儀涉及 3 位以上神將
- 科儀中有道兵渡劫失敗
- 科儀使用了替身道兵或輪值關口
- 科儀耗時或法力超出預估 50% 以上
- 法師明確要求

## 自動觸發法會錄

在大法主敕令中加入以下片段，科儀結束後自動生成法會錄：

```
科儀圓滿（或因天劫中止）後：
1. 彙整所有壇位的功德簿與天劫記錄
2. 依「法會錄」格式生成回顧報告
3. 存放於 [壇場根目錄]/postmortem/[日期]-[科儀名].md
4. 將行動項目摘要呈報法師
```

## 法會錄 vs 功過格

| 項目 | 功過格 | 法會錄 |
|------|--------|--------|
| 關注點 | 單一道兵的表現 | 整場科儀的運作 |
| 粒度 | 道兵級 | 科儀級 |
| 內容 | 評分與等第 | 流程回顧與經驗提煉 |
| 目的 | 優化符籙 | 優化科儀設計 |
| 觸發 | 每位道兵結束後 | 整場科儀結束後 |
