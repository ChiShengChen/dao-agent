# 功過格 ── Invocation Audit & Performance Scoring

每次開壇行法，皆入功過格。道兵的召喚、行法、圓滿或渡劫，一筆一筆記錄在冊。

---

## 壹、召喚記錄（Invocation Log）

每次敕令道兵，記錄以下資訊：

```markdown
## 召喚記錄 #[序號]

| 項目 | 內容 |
|------|------|
| 時辰 | [ISO 8601 timestamp] |
| 法號 | [道兵名] |
| 神將類型 | [獨行/千里眼/造化/監察/大法主/護法] |
| 天命 | [一句話描述任務] |
| 科儀 | [單獨行法 / 所屬科儀名] |
| 壇位 | [若在大醮中，標明第幾壇] |
| 符籙形制 | [飛符/鎮壇符/大醮科儀] |
| 加持 | [自省訣/通報訣/靜心訣/結界符/金剛戒/天眼通] |
| 法力預算 | [預估 tokens] |
| 適用戒律 | [TC-xxx, RC-xxx, CS-xxx] |
| 結果 | ✅ 圓滿 / ⚠️ 帶傷圓滿 / ❌ 渡劫失敗 / ⏸️ 法師介入 |
| 實耗法力 | [實際 tokens]（若可取得） |
| 耗時 | [duration] |
| 備註 | [特殊情況記錄] |
```

## 貳、功過考核（Performance Scoring）

科儀結束後，對每位道兵進行功過考核。

### 考核維度

| 維度 | 權重 | 滿分 | 評分標準 |
|------|------|------|---------|
| 天命達成 | 40% | 100 | 任務目標完成度 |
| 法力效率 | 20% | 100 | 實耗 vs 預估（越接近越好，超支扣分） |
| 品質 | 20% | 100 | 功德簿品質、格式正確性、無殘缺 |
| 戒律遵守 | 10% | 100 | 違規次數與等級（天劫 -30, 地劫 -10, 人劫 -3） |
| 渡劫能力 | 10% | 100 | 遇到天劫時的處理品質（優雅降級 +分，靜默失敗 -分） |

### 功過等第

| 等第 | 總分 | 評語 |
|------|------|------|
| 上上 | 90–100 | 功德圓滿，堪為表率 |
| 上中 | 80–89 | 功成行滿，略有瑕疵 |
| 中上 | 70–79 | 基本達標，尚有進步空間 |
| 中中 | 60–69 | 勉強合格，多處待改善 |
| 中下 | 40–59 | 未達標，需檢討符籙或任務拆分 |
| 下下 | 0–39 | 嚴重失敗，需徹底重新設計 |

### 考核奏摺格式

```markdown
# 功過考核

## 道兵資訊
- 法號：[名]
- 天命：[任務]
- 科儀：[所屬科儀]

## 各維度評分

| 維度 | 得分 | 說明 |
|------|------|------|
| 天命達成 | /100 | [具體說明] |
| 法力效率 | /100 | 預估 XK tokens，實耗 YK tokens |
| 品質 | /100 | [具體說明] |
| 戒律遵守 | /100 | [違規記錄，無則滿分] |
| 渡劫能力 | /100 | [天劫處理情況，無劫則滿分] |

## 加權總分：[X] / 100
## 等第：[上上/上中/中上/中中/中下/下下]

## 功過摘要
- 功（值得保留的做法）：
  - [具體表現]
- 過（需要改善的地方）：
  - [具體問題]

## 改善建議
- [針對「過」的具體改善方案]
```
